# YT List

A local-first YouTube playlist PWA.

## Features

- Add YouTube videos by dropping URLs or text anywhere in the app
- Manage multiple local playlists
- Play videos in a full-area YouTube embed
- Continue to the next video when playback ends
- Switch between sequence, shuffle, and repeat-one playback
- Move or hide the playlist panel
- Save playlists and settings in local storage

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The app is configured for GitHub Pages at `/yt-list/`.
